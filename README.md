# ML Practice Lab

A repository of machine learning practice exercises, experiments, and exploratory implementations created during my learning process.

## Overview

This repository contains my machine learning practice work.  
Its main purpose is to support learning through hands-on implementation, experimentation, and iteration.

This is not intended to be a polished portfolio repository.  
Some exercises may be incomplete or lightly documented, since the focus is on learning efficiently rather than refining every detail.

## Purpose

The goals of this repository are to:

- practice machine learning concepts through implementation
- experiment with different datasets, models, and workflows
- improve technical skills through repeated exercises
- document my learning progress in a structured way

## Structure

Each exercise is stored in its own directory.  
Depending on the exercise, a directory may include notebooks, source code, notes, or other related files.

## Exercise Notes

For each exercise, I include a short note with its name and a brief explanation.

- **mobile-prices-classification** — This exercise explores the performance of Naive Bayes, KNN, and SVM models on a dataset under different feature engineering settings.

## Note for Visitors

This repository reflects an active learning process.  
It should be viewed as a collection of practice-based and iterative machine learning work, not as a set of fully refined showcase projects.
